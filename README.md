# Digital Trading 
Projeto de landing page para o projeto "Digital Trading"

# Instalação 
1. Clone o repositório 
 ```sh
   git clone https://github.com/jgabrielleal/digital-trading.git
```

2. Navegue até a pasta 
```sh
cd digital-trading
```

3. Abra o arquivo "index.html"
```sh
start index.html
```
